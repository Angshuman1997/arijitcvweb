
# Portfolio Website for Designer

A modern portfolio website built with React and Create React App.

Original design available at https://www.figma.com/design/eMcBlqZlBtCq40NphFN6JX/Portfolio-Website-for-Designer.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm

### Installation

1. Clone the repository or extract the files
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server:
```bash
npm start
```

This will open the app in your browser at `http://localhost:3000`.

### Building for Production

To create a production build:
```bash
npm run build
```

To serve the production build locally:
```bash
npm run serve
```

### Technologies Used

- **React 18** - UI library
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **Radix UI** - Headless UI components
- **Lucide React** - Icon library
- **Create React App** - Build tooling

### Features

- Responsive design
- Modern UI components
- Tab-based navigation
- Portfolio showcase
- Contact information
